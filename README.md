# Fortune 500 ESG Knowledge Graph Site

财富 500 强 ESG 知识图谱网站

This folder contains only the public-facing static website.

此文件夹仅包含面向公开展示的静态网站内容。

It contains:

- `index.html`: site entry page
- `zh/`: Chinese pages
- `en/`: English pages
- `assets/css/`: stylesheet assets
- `assets/figures/`: bilingual chart images
- `.nojekyll`: GitHub Pages static-site marker

项目包含：

- `index.html`：网站首页入口
- `zh/`：中文页面
- `en/`：英文页面
- `assets/css/`：样式文件
- `assets/figures/`：中英文图表图片
- `.nojekyll`：GitHub Pages 静态站点标记
