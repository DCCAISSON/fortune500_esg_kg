Fortune 500 ESG Knowledge Graph

This site presents my ESG knowledge graph built from public Fortune 500 annual reports, sustainability reports, and carbon-related disclosures. The public version keeps the original 10-report demo and now foregrounds two axes: the standards/frameworks layer and the emissions-methodology/data-quality layer.

The site publishes structured outputs, graph files, visuals, and downloadable content only. It does not expose project source code or conversation history. The 500-report production track is being advanced in a separate batch pipeline.

Two priority axes on the homepage

Standards and Frameworks Layer

Identifies whether firms follow GHG, ISO, GB/T, GRI, TCFD, IFRS S2, ESRS/CSRD and related systems, then classifies them by standard, guideline, principle, assurance standard, and target-validation role.

Emissions Methodology and Data Quality Layer

Identifies primary data, secondary data, estimation/derivation, report disclosure, verification, assurance, validation, and activity classes such as stationary, mobile, fugitive, process, purchased electricity, and Scope 3 categories.

Current graph logic

Standards are modeled separately from standard-reference facts, so one framework can play different roles across reports.

Emission values are modeled separately from methodology facts, allowing each scope record to connect to data source, assurance stage, activity category, and calculation method.

Industry classification follows GB/T 4754-2017 at the section level to maintain a consistent cross-company comparison basis.

Activity tagging distinguishes explicit mentions from evidence-based mapping to support later human review and reinsertion.

This site contains presentation content only, not source code or conversation history.
